## 📌 프로젝트 소개

제로픽(ZeroPick)은 제로음식의  
**사진을 찍으면 OCR이 인식하여 식품 정보를 가져와서 영양성분과 원재료 정보를 분석**해주는 프로젝트입니다.  

이를 통해 사용자에게 **더 나은 식품 선택**을 돕고,  
**제로식품 시장의 성장**에 기여하고자 합니다.

---

## 🔍 주요 기능

- 📸 음식 사진 촬영 및 업로드
- 🧠 AI 기반 제로식품 인식 및 성분 분석
- 🍽️ 영양 성분 정보 시각화
- 🏷️ 제로 식품 여부 판별 및 추천
